# Microsoft IQ solution accelerator: Responsible AI FAQ  

## What is the Microsoft IQ Solution Accelerator?

This solution accelerator provides a single intelligence layer unifying enterprise data, knowledge and decisions from Work IQ implemented in Microsoft M365 Copilot, Foundry IQ and Fabric IQ.

## What can this Solution Accelerator do?  

The solution accelerator is ready to deploy for demonstration, adoption, and enhancement for supply chain disruption management. It provides an integrated and AI-powered end-to-end solution that enables faster, more informed decisions. This is achieved by unifying enterprise data, business knowledge, and execution workflows into a shared context. 

The solution accelerator illustrates how enterprises can move from early disruption signals to impact assessment and coordinated action. It brings together business data, supplier constraints, and execution workflows in one shared context. This ensures customers and sales are not affected, and revenues are not disrupted.
